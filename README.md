🎓 Sri Lakshmi College ERP App

A Flutter-based College ERP Mobile Application designed to manage academic activities digitally for students, faculty, and administrators.

---

📱 Features

🔐 Role-Based Authentication

- 👨‍💼 Admin Access
- 🎓 Student Access
- 👨‍🏫 Faculty Access

---

🎓 Student Dashboard

- 📊 Subject-wise Attendance
- 📈 Marks per Subject
- 📝 Assignments
- 🏏 Sports Updates
- 💰 Fees Status
- 📢 Notices

---

👨‍🏫 Faculty Dashboard

- Manage student attendance
- Select department & subjects
- Update academic records

---

👨‍💼 Admin Dashboard

- View student details
- View faculty details
- Manage departments and classes

---

🏫 Departments

- BCA
- BCOM
- BA

---
